Die Maven Befehle
-----------------

Eclipse Konfiguration neu erzeugen: `$ mvn eclipse:clean eclipse:eclipse`

Anwendung bauen: `$ mvn clean package`

Anwendung ausführen: `$ mvn initialize exec:exec`
